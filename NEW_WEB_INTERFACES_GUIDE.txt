🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════════════════════

❌ Issue: "Page not found" error
✅ Solution: Ensure FastAPI server is running (python run.py)

❌ Issue: "Authentication failed" error
✅ Solution: Clear localStorage and login again
   In browser console: localStorage.clear()

❌ Issue: Charts not displaying
✅ Solution: Check if Chart.js is loaded, refresh the page

❌ Issue: API data not loading
✅ Solution: Check if API endpoints are responding:
   http://localhost:8000/docs

❌ Issue: Slow performance
✅ Solution: Clear browser cache and data load
   In browser console: localStorage.clear()
